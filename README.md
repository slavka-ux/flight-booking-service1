# ✈️ SkyBooking - Сервіс замовлення авіаквитків

## 📌 Про проєкт

**SkyBooking** — це сучасний веб-сервіс для швидкого пошуку, бронювання та оплати авіаквитків. Платформа надає користувачам зручний інтерфейс для планування подорожей, порівняння цін від різних авіакомпаній та онлайн-оформлення квитків у кілька кліків.

Сервіс створений з метою спростити процес покупки авіаквитків, зробити його максимально прозорим, безпечним та доступним для кожного користувача незалежно від рівня технічної підготовки.

---

## 🎯 Основні цілі проєкту

- Автоматизація процесу пошуку та бронювання авіаквитків
- Забезпечення актуальною інформацією про рейси в реальному часі
- Створення зручного особистого кабінету для управління бронюваннями
- Впровадження безпечних онлайн-платежів
- Надання цілодобової підтримки користувачів

---

## 🚀 Основний функціонал

### Для користувачів:

- **Пошук рейсів** — за напрямком, датою, кількістю пасажирів та класом обслуговування
- **Фільтрація результатів** — за ціною, часом вильоту, тривалістю перельоту, кількістю пересадок та авіакомпанією
- **Детальна інформація про рейс** — час вильоту та прибуття, статус, тип літака, розклад
- **Вибір класу обслуговування** — економ, комфорт, бізнес, перший клас
- **Бронювання місць** — з автоматичним утриманням на визначений час (15 хвилин)
- **Онлайн-оплата** — банківськими картками з використанням захищених протоколів
- **Генерація електронного квитка** — у форматі PDF з унікальним PNR-кодом
- **Надсилання квитка на пошту** — автоматичне відправлення після оплати
- **Особистий кабінет** — перегляд історії бронювань, статусів рейсів та збережених напрямків
- **Відстеження змін цін** — сповіщення про зниження вартості на обрані напрямки
- **Скасування бронювання** — з можливістю часткового повернення коштів згідно з тарифами
- **Багатомовність** — українська та англійська мови інтерфейсу

### Для адміністраторів:

- Управління рейсами (додавання, редагування, видалення)
- Моніторинг бронювань та платежів
- Управління користувачами та їхніми правами
- Генерація звітів про продажі та завантаження рейсів
- Налаштування системи знижок та акцій
- Управління контентом (новини, статті, поради мандрівникам)

## 🛠 Технологічне рішення

### Архітектура

Сервіс побудований за принципами мікросервісної архітектури, що забезпечує:

- Високу масштабованість окремих компонентів
- Відмовостійкість системи
- Незалежне оновлення модулів
- Розподілене кешування даних

### Основні компоненти

| Компонент | Призначення |
|-----------|-------------|
| **API Gateway** | Маршрутизація запитів, аутентифікація, обмеження частоти запитів |
| **Сервіс пошуку** | Обробка пошукових запитів, кешування результатів, інтеграція з GDS |
| **Сервіс бронювання** | Управління бронюваннями, перевірка наявності місць |
| **Платіжний сервіс** | Обробка платежів, інтеграція з платіжними шлюзами |
| **Сервіс сповіщень** | Відправка email-листів, push-повідомлень, SMS |
| **Сервіс звітності** | Генерація статистичних звітів для адміністраторів |

### Інтеграції

- **GDS (Global Distribution Systems)** — отримання актуальної інформації про рейси, тарифи та наявність місць у реальному часі
- **Платіжні шлюзи** — LiqPay, Stripe, PayPal (підтримка основних платіжних методів)
- **Поштові сервіси** — SendGrid, AWS SES для надсилання квитків та сповіщень
- **Сервіси аналітики** — Google Analytics, Mixpanel для відстеження поведінки користувачів

## 📋 Етапи роботи з сервісом

1. **Реєстрація / Вхід** — створення облікового запису або авторизація через Google/Facebook
2. **Пошук рейсів** — заповнення форми з напрямком, датами, кількістю пасажирів та класом
3. **Вибір рейсу** — порівняння запропонованих варіантів за ціною, часом та умовами
4. **Введення даних** — заповнення інформації про пасажирів (ПІБ, дата народження, документи)
5. **Додаткові послуги** — вибір багажу, страхування, харчування, вибір місця в салоні
6. **Оплата** — введення платіжних даних та підтвердження транзакції
7. **Отримання квитка** — завантаження PDF-файлу та отримання копії на електронну пошту
8. **Управління бронюванням** — перегляд, зміна або скасування в особистому кабінеті

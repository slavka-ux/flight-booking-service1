**Flight Booking Service** – це повнофункціональний веб-сервіс для пошуку та бронювання авіаквитків.  
Проєкт включає:

- 🔍 **Пошук рейсів** за напрямком та датою
- 📝 **Бронювання** з додаванням кількох пасажирів
- 🔐 **Аутентифікація** (реєстрація, вхід, JWT)
- 💳 **Оплата** (тестовий режим: картка, Google Pay, Apple Pay)
- 👑 **Адмін-панель** для управління рейсами, бронюваннями та користувачами
- 🌐 **GitHub Pages** для статичного фронтенду
